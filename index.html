The NCIP-IX Job Portal is a web-based system built for the National Commission on Indigenous Peoples – Region IX to simplify and digitize their recruitment process. It connects job seekers and HR administrators in one platform, replacing manual application handling with a centralized online workflow.

Applicants can browse job openings, view requirements, and submit applications online with uploaded documents. Everything is handled digitally for faster and more organized submission.

On the admin side, the system allows secure management of job postings, applicant tracking, and recruitment status updates. It also includes a dashboard for monitoring applications and tools for reviewing and downloading applicant files efficiently.

OJTime Tracking

Role: Full Stack Developer

A web-based Daily Time Record (DTR) system built to replace manual paper logs with real-time tracking, PIN login, and PWA support.

OJT Time Tracking is a web-based Daily Time Record (DTR) system I built during my internship to replace manual paper logs and Excel encoding. It lets users digitally record time-in and time-out, automatically calculates total hours, validates entries, and generates a properly formatted DTR ready for printing and submission.

The system includes PIN-based login, real-time time logging, manual past entry encoding, attendance tracking, and automated DTR export with official layout and institutional details. It is also built as a Progressive Web App (PWA), so it can be installed on mobile devices and used even with unstable internet connection.

Role and Contribution

I built the entire system from scratch—handling UI/UX design, front-end development, core system logic, validation rules, Supabase database integration, PWA setup, and print-ready document layout. I also managed the system architecture, responsiveness across devices, and deployment on Vercel.
